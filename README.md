# Gerador Profissional de CV — Jose Victor Dromart

Este projeto é um **gerador de Currículos (CV) profissional** feito com **HTML, CSS e JavaScript**. Ele permite que o usuário preencha seus dados pessoais, acadêmicos e profissionais, visualize o CV em tempo real e exporte em PDF sem necessidade de banco de dados.

---

## **Funcionalidades**

* Preenchimento de **Informações Pessoais**: Nome, Data de Nascimento, Nacionalidade, Estado Civil, Endereço, Telefone, Email e NUIT.
* Campos para **Objetivo Profissional, Formação Acadêmica, Competências Técnicas, Experiência, Idiomas, Habilidades Pessoais, Projetos, Voluntariado e Disponibilidade**.
* **Validações automáticas**:

  * Telefone: 9 dígitos, começa com 82–87.
  * Email: deve conter `@`.
  * NUIT: 10 dígitos.
  * Data de nascimento: formato `dd/mm/aaaa`.
* **Pré-visualização em tempo real** do CV em formato profissional.
* **Exportação em PDF** com layout limpo, azul e branco.
* Interface moderna e responsiva, adequada para visualização em desktop e mobile.

---

## **Como usar**

1. Abra o site no navegador:
   [Link do GitHub Pages](https://seunomeusuario.github.io/repositorio/)
2. Preencha todos os campos obrigatórios do formulário.
3. Clique em **Pré-visualizar** para ver seu CV.
4. Clique em **Exportar PDF** para gerar o arquivo profissional.

---

## **Tecnologias utilizadas**

* **HTML5** — Estrutura do site
* **CSS3** — Estilo e animações
* **JavaScript** — Lógica do formulário e validações
* **html2pdf.js** — Biblioteca para gerar PDF

---

## **Notas**

* O projeto não utiliza **banco de dados**.
* Todos os dados são processados localmente no navegador do usuário.
* Design moderno e limpo, ideal para impressões e envio digital.
